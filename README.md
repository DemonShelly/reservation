# Hotel Reaservation 

## Demo
https://demonshelly.github.io/reservation/#/

## Discription
這是六角學院所舉辦的第二屆「前端精神時光屋大賽」的第六週題目，根據設計師[edith.hsu](https://z7x2c0v0b8.github.io/the_f2e_2nd/hotel_reservation.html)所撰寫成的訂房網站。

## Function
- 可以總覽不同房型的價格
- 進到不同房間的頁面後可點擊圖片放大查看
- 可以選擇訂房日期，網站會自動計算好價格呈現
- 訂房後該日期會顯示灰色，表示無法再次被預定
